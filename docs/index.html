Project Overview

This project implements a complete movie ticket booking and management system with advanced data analytics capabilities. It simulates real-world theater operations and provides business intelligence through interactive dashboards.

The system handles everything from movie catalog management, theater operations, booking processing, customer management, to comprehensive revenue analytics and visualization.

Technologies Used:
Python 3 SQLite Pandas & NumPy Plotly Matplotlib Seaborn

Key Features

Normalized Database

7 normalized tables with proper constraints, foreign keys, and indexes for optimal performance and data integrity.

Revenue Analytics

Comprehensive revenue analysis including daily trends, top movies, theatre performance, and customer segmentation.

Key Tables:

  • Movies: Movie details, ratings, production costs
  • Theatres: Theatre locations, facilities, capacity
  • Screens: Screen types (IMAX, 3D, 4DX, VIP), capacities
  • Shows: Show scheduling with dynamic pricing
  • Customers: Customer profiles with loyalty points
  • Bookings: Booking transactions with seat management
  • Payments: Payment processing and refunds
Sample Database Creation Code:
CREATE TABLE Movies (
    movie_id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    genre TEXT,
    duration_minutes INTEGER,
    rating DECIMAL(3,1),
    is_active BOOLEAN DEFAULT 1
);

CREATE TABLE Shows (
    show_id INTEGER PRIMARY KEY AUTOINCREMENT,
    movie_id INTEGER NOT NULL,
    screen_id INTEGER NOT NULL,
    show_date DATE NOT NULL,
    show_time TIME NOT NULL,
    price DECIMAL(10,2),
    available_seats INTEGER,
    FOREIGN KEY (movie_id) REFERENCES Movies(movie_id)
);

cd Movie-Ticket-Management-System

# 2. Install required packages

# 3. Run the Jupyter notebook
